🔑 API Key 申請教學

兩個進階功能需要免費 API Key：🤖 AI 家教（Gemini）和 🖼️ 單字動圖（Giphy）。

🔒 安全聲明：你的 Key 只存在自己的瀏覽器中，EnglishGo 不會傳送、儲存或收集你的 Key。請放心使用！

🤖 Part 1：Gemini API Key

  1. 前往 aistudio.google.com/apikey
  2. 登入 Google 帳號
  3. 點擊「Create API Key」
  4. 複製 Key
  5. 在 EnglishGo → AI 家教 → ⚙️ → 貼上 → 存

免費額度：每分鐘 15 次，對學習綽綽有餘。

🖼️ Part 2：Giphy API Key

  1. 前往 developers.giphy.com
  2. 註冊/登入
  3. Dashboard → Create an App → 選 API
  4. 填寫 App Name「EnglishGo」
  5. 複製 Key
  6. 在 EnglishGo → SRS 單字卡 → ⓘ → 貼上 → 存

免費額度：每小時 100 次搜尋。

❓ 常見問題

不申請可以用嗎？ 可以！15 個模組中有 13 個不需要 Key。

Key 會過期嗎？ 不會，除非你自己刪除。

真的免費嗎？ 真的，兩個都是免費方案。
