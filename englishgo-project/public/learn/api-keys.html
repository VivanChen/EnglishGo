🔒 安全與隱私

🛡️ 你的 Key 100% 安全
  • API Key 只儲存在你自己的瀏覽器中（localStorage）
  • Key 不會傳送到 EnglishGo 的伺服器（我們沒有伺服器！）
  • Key 不會被收集、記錄或分享給任何第三方
  • 你的瀏覽器直接和 Google / Giphy 通訊，EnglishGo 不經手
  • 清除瀏覽器資料時，Key 會一起被清除
  • 隨時可以在原平台（Google AI Studio / Giphy Dashboard）刪除或重新產生 Key

技術說明（給好奇的人）

EnglishGo 是一個純前端的 Single Page Application (SPA)，所有程式碼都在你的瀏覽器中運行：

  • Gemini API Key：存在 localStorage("eg_gemkey")，呼叫 API 時直接從瀏覽器發送到 generativelanguage.googleapis.com
  • Giphy API Key：存在 localStorage("eg_gifkey")，呼叫 API 時直接從瀏覽器發送到 api.giphy.com

EnglishGo 的原始碼是公開的，你可以隨時檢查我們的程式碼確認以上聲明。

❓ 常見問題

Q：不申請 Key 可以用 EnglishGo 嗎？

A：可以！SRS 單字卡、測驗、口說練習、打地鼠、配對翻牌、聽寫、文法、閱讀等 11 個模組中有 9 個不需要任何 Key。只有 AI 家教和 GIF 動圖需要。

Q：Key 會過期嗎？

A：Gemini Key 不會過期。Giphy Key 也不會過期。除非你自己到原平台刪除。

Q：換了電腦或瀏覽器怎麼辦？

A：因為 Key 存在瀏覽器的 localStorage 中，換設備需要重新輸入。建議把 Key 記在安全的地方（例如 Google Keep）。

Q：真的免費嗎？會不會被收費？

A：真的免費！Google Gemini 的免費額度對個人學習綽綽有餘。Giphy 的免費方案每小時 100 次搜尋，正常學習不可能用完。
