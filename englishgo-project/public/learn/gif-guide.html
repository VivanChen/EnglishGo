EnglishGo›單字動圖效果與設定

單字動圖不是必填，但能讓單字更好記

EnglishGo 的 SRS 單字卡預設會用內建圖片或表情符號。申請並貼上 Giphy API Key 後，系統會用目前單字自動搜尋適合的 GIF 動圖，讓孩子用畫面和動作一起記憶。

不申請也可以用

單字卡、發音、例句、測驗和遊戲都可以正常使用。沒有 Giphy Key 時，畫面會顯示內建圖片或表情符號。

申請後會看到什麼？

進入 SRS 單字卡時，每張卡會依目前單字搜尋動圖。例如 jump 可能顯示跳躍動作，cat 可能顯示貓的短動畫。

Key 放在哪裡？

Key 只會儲存在使用者自己的瀏覽器 localStorage。EnglishGo 不會把你的 Giphy API Key 存進資料庫。

申請 Giphy API Key

前往 developers.giphy.com，註冊或登入帳號。
進入 Dashboard，選擇 Create an App。
選擇 API 類型，App Name 可填 EnglishGo，用途可填 English vocabulary learning。
建立後複製 API Key。
回到 EnglishGo → 選等級 → SRS 單字卡 → 點右上角 說明 / 動圖設定 → 貼上並儲存。
提醒：Giphy 免費額度對一般學習已足夠，但仍受 Giphy 官方政策與流量限制影響。

先測試你的 Key

常見問題

一定要申請嗎？ 不需要。這是加分功能，不是核心功能。

要輸入圖片網址嗎？ 不用。只要貼上 Giphy API Key，EnglishGo 會依照目前單字自動找圖。

為什麼有些單字沒有 GIF？ Giphy 不一定每個英文單字都有適合兒童學習的結果，所以 EnglishGo 會保留內建圖片作為備援。

可以刪除 Key 嗎？ 可以。在 SRS 單字卡的設定欄把 Key 清空後儲存即可。
